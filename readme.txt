Welcome to I-am-core-Hieu project:

It is an interactive console application that allows you to create, delete, update and search identities. 
Read to the User Guide.pdf for further help.
This application was build using JDK 1.8

Step to run the application

Setup the database:

Download apache derby client
Create an instance of derby name it instance name =sample with username="hieu" and password="hieu".
run /db-derby-10.13.1.1-bin/bin/startNetworkServer
Go to I-am-core-Hieu/sql run create_tables.sql on your instance sample
Run the application:

Go to I-am-core-Hieu/src/fr/epita/iam/launcher execute ConsoleLauncher.java
